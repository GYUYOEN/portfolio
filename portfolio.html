  • [프로젝트 설명]
    hello, my life는 인스타그램 클론 코딩을 한 것으로써 기능은 인스타그램과 비슷합니다. 하지만 인스타그램을 써보고 대중적으로 많이 알려진 인스타그램에서 있었으면 편리할 것 같은 기능들을 새롭게 구현해보고 싶었습니다.

    팀프로젝트 때 사용했던 spring security를 이용해서 로그인과 회원가입을 진행하였습니다. 팀프로젝트 당시 놓쳤던 부분이나 시간이 없어 미처 사용하지 못했던 부분들을 보안하기 위해 다시한번 사용해봤습니다.

    또한 개발자 블로그를 알게 되면서 블로그를 써보게 된 계기가 되었고 블로그를 작성하면서 만들어본 첫 프로젝트입니다. 블로그를 통해 기록의 중요성을 알게되었고 블로그가 얼만가 개발하는데 도움이 되는지 배웠습니다.
    프로젝트 기간 :
    프로젝트 방식 : Spring Boot
    사용 프로그램 : Intellij, VSCode, MariaDB, Gradle, Github
    사용 언어 : Java, Html, Css, JavaScript, Jpa, bootstrap
    Github 주소 : https://github.com/GYUYOEN/instagram
  • [프로젝트 설명]
    workforus는 회사나 학업에서 자주 사용하는 그룹웨어 시스템입니다. 일정관리, 메일, 메신저, 근태관리, 게시판, 주소록의 기능이 있습니다. 제가 담당한 기능은 로그인, 회원가입, 내정보관리, 메신저 입니다.

    로그인과 회원가입은 spring security룰 사용하였습니다. 로그인과 회원가입의 UI & UX는 기존에 있던 디자인을 사용하였지만 메신저는 직접 디자인을 구현하였습니다. 메신저 구현은 WebSocket와 stomp를 이용함으로써 웹통신이 어떻게 이루어지는지 아는 중요한 계기가 되었습니다.

    학원에서 진행한 팀프로젝트이자 저의 첫 프로젝트입니다. 팀원은 학원 초반에 들어간 스터디모임 맴버들과 진행하였습니다. 팀프로젝트를 하면서 혐업이 무엇이고 협업에서 Github 관리는 어떻게 이루어지는지고 어떻게 진행해야지 편리한지 배웠습니다. 첫 프로젝트인 만큼 욕심도 컸지만 시간이 정해져 있어 욕심을 못이룬 아쉬움이 컸습니다.
    프로젝트 기간 :
    프로젝트 방식 : Spring MVC
    사용 프로그램 : Eclipse, Oracle, Maven, Github
    사용 언어 : Java, Html, Css, JavaScript, MyBatis, bootstrap, WebSocket, Stomp
    Github 주소 : https://github.com/hong-yura/WorkForUs-Project
